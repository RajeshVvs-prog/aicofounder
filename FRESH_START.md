# 🚀 FRESH START - Clean Setup

## What This Project Is:
An AI Co-founder app built in Google AI Studio. It helps validate startup ideas using Gemini AI.

## Architecture:
- **Frontend**: React + Vite (Port 3000) - The UI you see
- **Backend**: Express (Port 3001) - Handles Gemini API calls
- **API Key**: In `backend/.env` file

---

## Step-by-Step Fresh Start:

### 1. Make Sure Everything is Stopped
Press `Ctrl+C` in any terminal that's running

### 2. Start the App
Open terminal in this folder and run:
```bash
npm run dev
```

### 3. Wait for Both Servers
You should see:
```
✅ Backend running on http://localhost:3001
VITE v6.4.1  ready in 345 ms
➜  Local:   http://localhost:3000/
```

### 4. Open the App
**IMPORTANT**: Open this URL in your browser:
```
http://localhost:3000
```

**NOT** http://localhost:3001 (that's just the backend API)

### 5. Test It
- Click "Get Started"
- Choose "Idea Generation"
- Fill in the form
- Click "Generate Ideas"

---

## If You See Errors:

### Browser shows "Something went wrong":
1. Open browser console (Press F12)
2. Look for red error messages
3. Send me the error

### Terminal shows errors:
1. Look at the terminal output
2. Find lines that say "error:" or "Error:"
3. Send me those lines
